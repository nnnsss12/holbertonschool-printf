# _printf Project

## Description
A custom implementation of the C `printf` function for handling characters, strings, and percent signs.

## Functionality
- `%c`: Prints a character.
- `%s`: Prints a string.
- `%%`: Prints a percent sign.

## Requirements
- Compiled with: `gcc -Wall -Werror -Wextra -pedantic -std=gnu89`
- Style: Betty style.
